# 课堂电子宠物（ClassBuddy）网站产品计划书

## 1. 产品概述
ClassBuddy 是一款专为中小学教师设计的课堂参与度管理工具。通过将课堂行为游戏化（Gamification），教师可以将抽象的课堂表现转化为可视化的电子宠物成长过程，从而激发学生的学习动力。

## 2. 核心功能模块

### 2.1 学生与宠物管理
* **学生录入**：支持教师手动输入姓名或通过 Excel 批量导入名单。
* **宠物领养**：学生可从初始宠物品种中选择其一（如：救援犬系列、光之巨人系列、魔法精灵系列）。
* **身份绑定**：每个姓名对应一个唯一的宠物 ID 和成长档案。

### 2.2 能量值与奖励系统
* **能量获取机制**：
    * **积极发言**：+10 能量
    * **作业优秀**：+20 能量
    * **小组协作**：+15 能量
    * **遵守纪律**：+5 能量
* **操作界面**：教师端提供“一键加分”按钮，支持多人同时选定奖励。

### 2.3 进化系统（三阶段）
宠物根据累计能量值自动改变形态，提升学生的成就感。

| 进化等级 | 能量阈值 | 视觉表现描述 | 心理暗示 |
| :--- | :--- | :--- | :--- |
| **阶段一：幼年期** | 0 - 50 | 形象呆萌、体型较小、无特殊装备 | 充满潜力的初学者 |
| **阶段二：成长期** | 51 - 150 | 体型增大、穿戴初级救援装备或散发微光 | 表现稳健的进步者 |
| **阶段三：完全体** | 151+ | 拥有终极机甲/披风、华丽特效、威武姿态 | 课堂模范/传奇英雄 |

### 2.4 数据展示与看板
* **荣誉看板（Hall of Fame）**：全班宠物按能量值从高到低排列，展示实时排名。
* **进化动画**：当宠物达到进化临界点时，触发全屏“形态觉醒”动画。

## 3. 技术架构建议
* **前端**：React.js 或 Vue.js (响应式设计，适配投影仪与平板)。
* **后端**：Firebase 或 Supabase (提供实时数据库支持，确保数据即时更新)。
* **图像处理**：
    * **固定素材库**：预设高品质角色图集。
    * **AI 动态生成**：集成 Gemini/Imagen API，根据学生姓名和等级动态绘制独一无二的宠物形象。

## 4. 商业化方案
* **基础版 (Freemium)**：支持基础学生人数和预设宠物品种，免费供个人教师使用。
* **专业版 (Pro)**：解锁 AI 动态形象生成、导出成长报告、自定义奖励标准。
* **校园版 (Enterprise)**：学校统一管理，数据跨班级同步，支持全校范围的“英雄联赛”。

## 5. 开发路线图 (Roadmap)
1.  **Phase 1 (MVP)**：完成学生录入、手动加分及宠物静态图片切换。
2.  **Phase 2 (UX)**：加入平滑的进化过渡动画及全屏庆祝特效。
3.  **Phase 3 (AI Integration)**：上线 AI 辅助生图功能，实现宠物的个性化定制。
4.  **Phase 4 (Social)**：增加班级共同任务（如：全班总能量达标解锁新背景）。
